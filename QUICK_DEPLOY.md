## âš¡ Fast Deploy: Railway (30 Minutes)

Let's deploy your trading AI to Railway with your domain.

### Step 1: Prepare Repository (5 min)

```bash
cd /Users/silasmarkowicz/trading-ai-working

# Create railway.json
cat > railway.json <<EOF
{
  "$schema": "https://railway.app/railway.schema.json",
  "build": {
    "builder": "NIXPACKS"
  },
  "deploy": {
    "startCommand": "streamlit run src/dashboard/unified_dashboard.py --server.port=\$PORT --server.address=0.0.0.0",
    "restartPolicyType": "ON_FAILURE",
    "restartPolicyMaxRetries": 10
  }
}
EOF

# Create Procfile (backup start method)
echo "web: streamlit run src/dashboard/unified_dashboard.py --server.port=\$PORT --server.address=0.0.0.0" > Procfile

# Make sure requirements.txt has streamlit
grep -q "streamlit" requirements.txt || echo "streamlit>=1.28.0" >> requirements.txt
```

### Step 2: Deploy to Railway (10 min)

1. **Go to**: https://railway.app
2. **Sign up** with GitHub
3. **Click**: "New Project"
4. **Select**: "Deploy from GitHub repo"
5. **Choose**: trading-ai repository
6. **Wait**: ~3-5 minutes for build

### Step 3: Add Your Domain (5 min)

**In Railway Dashboard**:
1. Click your project â†’ Settings
2. Go to "Domains" section
3. Click "Custom Domain"
4. Enter: `trading.yourdomain.com` (or whatever subdomain)
5. Copy the CNAME record shown

**In Your Domain Provider** (GoDaddy/Namecheap/Cloudflare):
1. Go to DNS settings
2. Add CNAME record:
   - **Name**: `trading` (or your subdomain)
   - **Value**: `xxxxx.up.railway.app` (from Railway)
   - **TTL**: Auto or 3600
3. Save

**SSL Certificate**: Auto-generated by Railway (takes ~5 min)

### Step 4: Set Environment Variables (5 min)

**In Railway Dashboard â†’ Variables**:

```bash
# Critical (set these now)
ALPACA_API_KEY=your_alpaca_key
ALPACA_SECRET_KEY=your_alpaca_secret
ENVIRONMENT=production
PYTHONUNBUFFERED=1

# Optional (add later when ready)
ETH_PRIVATE_KEY=your_eth_key_when_ready
BASE_RPC_URL=https://mainnet.base.org
DATABASE_URL=${{Postgres.DATABASE_URL}}  # Auto-added if you add PostgreSQL
REDIS_URL=${{Redis.REDIS_URL}}  # Auto-added if you add Redis
```

**âš ï¸ Security**: Variables are encrypted at rest on Railway

### Step 5: Add Database (Optional, 5 min)

**If you want real data (not fake)**:

1. In Railway, click "+ New"
2. Select "PostgreSQL"
3. Wait for provisioning (~1 min)
4. Railway auto-connects it (DATABASE_URL set automatically)

**Cost**: +$5/month for PostgreSQL

### Step 6: Verify Deployment (2 min)

